<p><strong>MasterChef Foods</strong><br>MasterChef Foods is an eCommerce Online Food Ordering Application.<br><br><strong>Overview</strong><br>This project works is aimed for developing an efficient food ordering system that can be used in the food industry which can help the restaurants to quickly and easily manage daily operational task as well as improve the dining experience of customers.<br><br><strong>Description</strong><br>&ldquo;MasterChef Food&rdquo; is a simple online Food ordering app. This app consist of functionalities like-<br> Login- A user can login by using Mobile number or email id. Every user need to &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;login first to access this application.<br> Once the user login then a homepage appears on the screen which consists a search &nbsp; &nbsp; &nbsp; &nbsp;bar, navigation header, toolbar, different categories of food menus along with veg &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;and non-veg sections. Using of search bar user can search any dishes, using &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;navigation header user can check order details, order history etc. and it also have a &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;logout button by which user can logout from the application. Toolbar have a &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;notification icon which show numbers of unread notifications and a cart icon which &nbsp; &nbsp; &nbsp; &nbsp;shows numbers of menu items added to cart. Veg and Non-veg sections shows &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;available veg and non-veg items.<br> Now user need to add menu items into cart before order. User can check his or her &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;cart which display items individually along their price and descriptions. It also &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;display total amount and an order button.<br> After clicking on the order button from cart user move to the next screen which &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;takes user details like user name, delivery address, mobile number etc. This screen &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;provide a button to proceed to the next activity which shows order confirmation like &nbsp; &nbsp; &nbsp;user name, delivery address, email id, booked items, total payable amount etc.<br> Finally user move to the payment activity, hear user can pay using different payment &nbsp; &nbsp; &nbsp;methods.<br><br><strong>Facility</strong><br>This is the online Food ordering app, which contains many facilities. Those facilities are clarified below.<br> &nbsp;Makes the ordering process easier.<br> &nbsp;Efficient customer and order management.<br> &nbsp;Free and cheap marketing. Having a strong online presence simply means being in &nbsp; &nbsp; &nbsp; &nbsp;front of your customers 24X7 without having to pay for expensive media advertising &nbsp; &nbsp; and billboards.<br> &nbsp;Better customer&rsquo;s data. Who are your regular customers? Which food items are &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; popular? Are they aware of the promotions and offers on the website? Do they &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;prefer ordering from an app? These and many other related questions can be &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;answered using analytics and insights provided by a robust online ordering system.<br> &nbsp;You can reach thousands of people at a time.<br> &nbsp;When customers can browse the menu without feeling rushed, they tend to spend &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; more time deciding what to order. The math is simple: the more time they have to &nbsp; &nbsp; &nbsp; &nbsp; &nbsp;look over, the more they are likely to order.<br> &nbsp;From meetings to crowded areas, there are times when one may not be able to make &nbsp; &nbsp; &nbsp; a phone call to order food. Online ordering allows customers to order anytime, &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; &nbsp; anywhere using their mobiles, tablets or other handheld devices.<br><br><strong>Built with</strong><br>Android Studio<br>Java<br>Firebase Firestore<br><br><br></p>
